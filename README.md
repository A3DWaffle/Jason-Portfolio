# Jason Anderson Portfolio

This is a simple personal portfolio website for Jason Anderson, a Computer Science student interested in programming, game design, developer tools, and game AI scripting.

## Live Site

Once GitHub Pages is enabled, the site will be available at a URL like:

`https://A3DWaffle.github.io/Jason-Portfolio/`

## About This Portfolio

The portfolio highlights selected game-related programming and tooling work, including:

- Expanding Fronts AI Tools, a published Visual Studio Code extension
- Expanding Fronts / Star Wars: Galactic Battlegrounds AI scripting work

## Technologies Used

- HTML
- CSS
- Responsive web design

This version is intentionally simple so it can be hosted easily with GitHub Pages.

## How to View Locally

1. Download or clone this repository.
2. Open `index.html` in a web browser.

No build tools, npm install, or server setup are required.
