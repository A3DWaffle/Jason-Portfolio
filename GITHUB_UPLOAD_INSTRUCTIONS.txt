GitHub Repo Update Instructions
===============================

Recommended repo name:
Jason-Portfolio

Files in this package:
- index.html
- README.md
- .gitignore
- GITHUB_UPLOAD_INSTRUCTIONS.txt

To upload these files to a new repo:
1. Open the GitHub repository.
2. Click "Add file".
3. Click "Upload files".
4. Drag in the files from this folder.
5. Click "Commit changes".

To update files after they already exist:
1. Open the file on GitHub, such as index.html.
2. Click the pencil icon to edit.
3. Paste the updated file contents.
4. Click "Commit changes".

To turn on GitHub Pages:
1. Open the repository on GitHub.
2. Go to Settings.
3. Click Pages in the left sidebar.
4. Under "Build and deployment", set Source to "Deploy from a branch".
5. Branch: main
6. Folder: /root
7. Click Save.

Your site should appear at a URL like:
https://A3DWaffle.github.io/Jason-Portfolio/

It may take a minute or two for the site to appear after enabling Pages.
